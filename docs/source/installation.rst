Installation
============

Requirements
------------

**pythonbible** requires Python and is tested with the following versions:

* 3.10
* 3.11
* 3.12
* 3.13
* 3.14

**pythonbible** has no other dependencies.

How to Install
--------------

To use **pythonbible**, first install it using pip:

.. code-block:: console

    $ python3 -m pip install pythonbible
